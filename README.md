# QuizPulse

A Rust + WebAssembly quiz app for classic novels, starting with **Middlemarch** by George Eliot.

Built with [`wasm-pack`](https://github.com/rustwasm/wasm-pack), webpack 5, and vanilla JS.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)
- Node.js (v16+) and npm

## Running locally

**1. Build the WebAssembly package** (from the project root):

```bash
wasm-pack build
```

**2. Install frontend dependencies** (first time only):

```bash
cd www
npm install
```

**3. Start the dev server:**

```bash
npm run start
```

Open [http://localhost:8080](http://localhost:8080).

## Project structure

```
src/          Rust library — quiz engine compiled to WASM
pkg/          Generated by wasm-pack (do not edit)
www/          Web frontend (webpack 5, HTML, JS)
tests/        Rust unit tests
```

## Running tests

```bash
# Rust unit tests
cargo test

# WASM tests in a headless browser
wasm-pack test --headless --firefox
```
